this is a Basic network designed using Ciso Packet Tracer for an expantion in Stmary's University which allows for connections between devices across 5 Main buildings
buildings:

• Building A: Administration, Finance, and IT

• Building B: Classrooms and Engineering Labs

• Building C: Library and Student Services

• Building D: Research Hub with HPC and IoT Labs


• Building E: Student Accommodation and Gym

The university seeks a modern, secure, and smart network infrastructure that
supports:

• Wired and wireless access across all buildings

• Segmented internal networks for staff, students, research, and admin

• IoT-enabled smart systems
• Monitoring and alerting systems for IT support

• Secure handling of sensitive research and student data

• Ethical compliance with regard to surveillance and data privacy

You are tasked with designing and implementing this solution using Cisco Packet

Tracer and explaining your design in a comprehensive report.
